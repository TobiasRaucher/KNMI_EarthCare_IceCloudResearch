# KNMI_EarthCare_IceCloudResearch
Bachelor's thesis research at KNMI analysing ice cloud properties from the EarthCARE satellite's ATLID lidar instrument. Processes Level-2 HDF5 data products (ATL_TC, AC_TC, ATL_EBD, ATL_ICE) into gridded NetCDF outputs and produces global distribution, microphysics, and optical property figures.
